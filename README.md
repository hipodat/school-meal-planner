# 급식 식단표 도우미

나이스(NEIS) 급식 데이터로 월간 식단표를 만드는 Next.js 앱입니다.

## 실행 방법

1. Node.js 18 이상 설치 (https://nodejs.org)
2. 이 폴더에서 터미널 열고:

```bash
npm install
npm run dev
```

3. 브라우저에서 http://localhost:3000 접속
4. 학교명을 검색해서 선택하면 해당 월 급식이 자동으로 뜹니다.

## 인증키

- `.env.local` 파일에 `NEIS_KEY` 로 넣어두었습니다.
- 이 파일은 `.gitignore` 에 있어 깃허브에 올라가지 않습니다. 공개 저장소에 키를 노출하지 마세요.

## 기능

- 학교 검색 → 월간 달력에 급식 자동 표시
- 요리명 뒤 알레르기 번호(1~19) 자동 파싱·배지 표시
- 칼로리·영양정보 표시 (편집 화면)
- 날짜별 메뉴 추가/삭제 (현재는 브라우저 세션 내 임시 저장)
- '이번 달 적게 나온 메뉴' 추천
- 인쇄·PDF, 엑셀(CSV) 내보내기
